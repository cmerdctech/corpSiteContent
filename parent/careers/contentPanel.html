SMART Systems Development

A Cooperative Project between ESV Regions 1-5


Software Tester


SMART Systems Development provides financial and human resource software that is currently deployed in more than 300 school districts throughout Minnesota. This position will be testing the programs to ensure that the software will perform well in a production situation.


Responsibilities:

  • Creation and implementation of test scenarios in which tests are carried out

  • Keeping track of standards

  • Performance testing on functionality of software

  • Reporting defects and problems that occurred during the test process

  • Using a tracking system to track reported problems with software

  • Documentation and training of new software processes and items ready for release

Preferred Qualifications: 

  • Knowledge of modern test methodologies

  • Good analytical and communication skills

  • Knowledge of software development methodologies

  • Knowledge in concepts of databases and/or basic knowledge of SQL Software Testing

  • Understanding/knowledge of QA processes and methods used in software development
