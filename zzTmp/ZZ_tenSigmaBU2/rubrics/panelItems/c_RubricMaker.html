Rubrics in the Online RubricMaker


The RubricMaker is an online software program that allow users to access existing rubrics and design new rubrics that communicate expectations, focus instruction, structure assessment, and report progress in a more measurable manner.

The RubricMaker allows users to:
  • Search for rubrics
  • Edit rubrics
  • Set a variety of scales
  • Print rubrics
  • Design new rubrics
