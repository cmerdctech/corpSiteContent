Transition Plan


The Transition Plan allows the user to summarize on one page all the student needs to learn, accomplish, or complete during the transition years.

The user develops the plan to define the student's ( 1) postsecondary goals, ( 2) transition strengths, and (3) transition priority needs.

The Transition Plan is used as the foundation for writing, presenting, discussing, and adapting the IEP

Using the plan involves the student and parents more in the transition process.

Using the plan increases the quality of the IEP and reduces IEP writing time.
